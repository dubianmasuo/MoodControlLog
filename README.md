# 自我感受记录网页应用

## 1. 应用概述

### 1.1 应用名称
每日状态轻记录

### 1.2 应用描述
- 一个专为焦虑或抑郁用户设计的自我感受记录网页应用，通过简单易行的记录方式帮助用户重建自我连接，找回生活的控制感和意义感。
= 记录是重拾感受和控制感的最佳方式之一。它能帮助您将脑海中混乱发散的思绪“外化”到纸上或屏幕上，从而理清头绪、梳理情绪。
### 1.3 应用说明
- 记录模块设计来源于与Gemini的心理咨询，适配移动端：
- I. 状态量化与觉察 (情绪聚焦)： 捕捉情绪、能量和生理状态（睡眠）。
- II. 积极捕捉与掌控 (行动聚焦)： 采纳积极心理学（三件开心的事）和控制感（自爱行动）。
- III. 觉察与反思 (深层探索)： 探索情绪触发点和消费行为，进行深层反思。
- IV. 明日展望 (减少拖延)： 设置明日的小目标，提供前瞻性的控制感。

## 2. 功能需求

### 2.1 基础信息模块
- 日期选择器：支持日期选择、前一天/后一天导航、快速回到今天
- 日期状态显示：显示当前日期记录状态（是否有记录）
- 自动保存显示：显示最后一次保存时间
- 编辑权限控制：限制未来日期编辑，保护用户免受预设压力

### 2.2 当下感受捕捉模块
- 情绪评分：1-10分滑块评分，1=非常低落，5=平静，10=非常快乐
- 能量评分：1-10分滑块评分，1=疲惫无力，5=正常，10=充满活力
- 睡眠记录：入睡时间和醒来时间选择器（必填项）
- 实时评分显示：滑块上方显示当前评分数值

### 2.3 美好时刻记录模块
- 今日最开心三件事：三个文本域，支持具体美好时刻记录
- 今日掌控感：记录主动完成的事情
- 今日自爱行动：记录为身体/心理健康采取的行动

### 2.4 情绪分析模块
- 情绪触发点：记录导致情绪低落的具体事件或想法
- 自救行为：记录如何从低落中恢复
- 消费记录：总消费额、情绪性消费、投资性消费分类记录

### 2.5 未来规划模块
- 明天三件小事：三个文本域，记录具体可执行的明日任务
- 任务格式指导：提供「动词+对象+时间/数量」的填写建议

### 2.6 数据可视化模块
- 近7天心情图表：展示情绪和能量趋势变化
- 图例说明：情绪评分（蓝色）和能量评分（绿色）区分显示

### 2.7 数据管理模块
- 数据导出：支持Markdown、CSV、JSON格式导出（单天/全部）
- 数据导入：支持CSV/JSON文件导入
- 打印功能：支持打印当前页面或另存为PDF
- 备份提醒：记录达到一定数量时提醒用户备份
- 存储监控：显示当前本地存储使用情况

### 2.8 日历浏览模块
- 月份导航：支持上个月/下个月切换
- 记录标记：已记录日期在日历上标记显示
- 快速跳转：点击日期直接跳转到对应记录页面

## 3. 技术特性

### 3.1 离线优先
- 纯离线应用，不依赖网络连接
- 数据完全存储在本地浏览器中
- 无任何用户数据追踪

### 3.2 数据安全
- 数据本地隔离存储
- 提供多重备份机制
- 清除缓存或更换设备会丢失数据，强烈建议定期备份

### 3.3 自动化功能
- 自动保存机制
- 日期默认为今天
- 滑块评分实时显示数值
- 字数统计和行数显示

## 4. 设计风格

### 4.1 配色方案
- 主色调：温和的蓝绿色系（#4f8ef5），营造安全感和治愈感
- 辅助色：清新的薄荷绿（#10b981）用于积极情绪
- 背景色：浅灰色（#f0f2f5）确保视觉舒适度
- 强调色：橙黄色（#ffcc00）用于重要提醒

### 4.2 视觉细节
- 圆角设计：统一使用8px圆角，减少尖锐感
- 柔和阴影：轻微阴影效果增加层次感
- 滑块控件：自定义滑块样式，触感友好
- 按钮样式：扁平化设计配合微妙渐变效果

### 4.3 布局特点
- 卡片式布局：每个功能模块独立成卡片\n- 响应式设计：适配桌面端和移动端
- 空间排版：充足的留白空间，降低视觉负担
